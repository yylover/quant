# ADX优化策略V2说明

## 策略原理
ADX优化策略V2采用全新的策略设计理念，结合多周期均线确认大趋势，使用ADX指标确认入场时机，实现更可靠的趋势跟随。

## 策略逻辑

### 大趋势定义
- **上升趋势**：20日均线 > 50日均线 > 100日均线
- **下降趋势**：20日均线 < 50日均线 < 100日均线

### 买入条件
1. 大趋势向上（20日 > 50日 > 100日均线）
2. ADX > 18（趋势存在）
3. +DI > -DI（上升趋势确认）
4. 价格在20日均线上方

### 卖出条件
1. 大趋势向下（20日 < 50日 < 100日均线）
2. ADX > 18（趋势存在）
3. -DI > +DI（下降趋势确认）
4. 价格在20日均线下方

## 策略参数
- **ADX周期**：14
- **均线周期**：20日、50日、100日
- **趋势阈值**：ADX > 18

## 回测结果
- **策略总收益率**：21.48%
- **基准收益率**：100.90%
- **超额收益**：-79.42%
- **年化收益率**：3.83%
- **最大回撤**：-16.57%
- **胜率**：51.97%
- **盈亏比**：1.06

## 策略特点
1. **多周期确认**：使用三条均线确认大趋势
2. **趋势优先**：在大趋势确定的情况下保持持仓
3. **信号过滤**：ADX指标过滤掉震荡市场的假信号
4. **稳健可靠**：避免频繁交易，提高信号质量

## 适用市场环境
- 适合明显趋势市场
- 在震荡市场中保持观望
- 对趋势反转反应及时

4. 优化均线周期组合
