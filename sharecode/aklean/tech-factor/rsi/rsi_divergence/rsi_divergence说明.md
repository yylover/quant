# RSI背离交易系统

## 策略说明

### 核心思路
利用RSI指标与价格的背离现象来识别潜在的反转信号。

### 系统结构
- **RSI周期**: 14天
- **背离检测周期**: 5天

### 交易逻辑
1. **看涨背离**: 价格创新低，但RSI未创新低，预示可能反转上涨
2. **看跌背离**: 价格创新高，但RSI未创新高，预示可能反转下跌
3. **买入信号**: 出现看涨背离时买入
4. **卖出信号**: 出现看跌背离时卖出
