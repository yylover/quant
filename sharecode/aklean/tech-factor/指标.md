量化里常说的**技术指标因子**，基本都是从价格、成交量衍生出来的。下面给你列一份**最常用、实战最多**的清单，按类别整理，方便你直接拿去做因子研究。

# 一、趋势类（判断涨跌方向）
1. **均线类 MA / EMA**
   - 5日、10日、20日、60日均线
   - 常用因子：价格在均线上方/下方、均线多头排列
2. **MACD**
   - DIF、DEA、MACD柱、金叉死叉
3. **布林带 BOLL**
   - 上轨/中轨/下轨，是否突破轨道
4. **动量指标 MOM**
   - 近N日涨跌幅、收益率累计
5. **平均趋向指标 ADX**
   - 判断趋势强弱，ADX越大趋势越强

# 二、反转类（抓高低点）
1. **RSI 相对强弱指标**
   - 超买（>70）、超卖（<30）
2. **KDJ**
   - K/D/J 值，高位死叉、低位金叉
3. **CCI 顺势指标**
   - 突破 ±100 视为极端行情
4. **WR 威廉指标**
   - 判断是否处于高位/低位
5. **乖离率 BIAS**
   - 股价偏离均线的程度，偏离过大容易反转

# 三、成交量类（看资金强度）
1. **成交量 / 换手率**
   - 放量上涨、缩量下跌
2. **OBV 能量潮**
   - 累计成交量，看资金进出方向
3. **VR 成交量比率**
   - 上涨量/下跌量，判断市场热度
4. **VMA 量均线**
   - 均量线金叉/死叉

# 四、波动与波动率类
1. **ATR 平均真实波幅**
   - 衡量波动大小，用于止损、仓位管理
2. **历史波动率 HV**
   - 收益率标准差
3. **振幅**
   - (最高价-最低价)/开盘价

# 五、量价结合类（量化很爱用）
1. **AMOUNT 成交额**
2. **量价相关性**
   - 价格与成交量的相关系数
3. **上涨放量/下跌缩量**
   - 强趋势特征

# 六、简单但非常强的“朴素因子”
很多量化私募不用花哨指标，反而用这些：
- 近N日收益率（动量）
- 近N日收益标准差（波动率）
- 换手率、成交额
- 开盘/收盘/最高/最低比价
- 涨跌停次数、连板次数

---

# 简单总结
- 做**趋势、动量**：MA、MACD、ADX、MOM
- 做**反转、抄底逃顶**：RSI、KDJ、CCI、BIAS
- 做**资金强度**：成交量、换手率、OBV
- 做**风控、止损**：ATR、波动率
