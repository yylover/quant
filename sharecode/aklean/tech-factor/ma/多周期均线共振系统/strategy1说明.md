# 多周期均线共振交易系统（策略1）

## 策略说明

### 核心思路
结合不同时间周期的均线，形成共振信号，提高交易成功率。多周期共振可以有效过滤假信号，捕捉真正的趋势机会。

### 系统结构
- **快线**：5日均线（短期趋势）
- **中线**：20日均线（中期趋势）
- **慢线**：60日均线（长期趋势）
- **超慢线**：120日均线（大趋势）

### 开仓条件（必须同时满足）
1. **超慢线向上**（大趋势确认）：120日均线斜率 > 0
2. **慢线向上**（长期趋势向上）：60日均线斜率 > 0
3. **中线上穿慢线**（中期趋势转强）：20日均线上穿60日均线
4. **快线上穿中线**（短期买入信号）：5日均线上穿20日均线
5. **价格站在所有均线上方**：价格 > 5日、20日、60日、120日均线

### 平仓条件（满足任一即可）
1. **快线下穿中线**（短期趋势反转）：5日均线下穿20日均线
2. **价格跌破中线**（止损）：价格 < 20日均线
3. **慢线向下**（长期趋势转弱）：60日均线斜率 < 0
