# 波动率自适应均线交易系统（策略3）

## 策略说明

### 核心思路
根据市场波动率自动调整均线参数。在高波动率环境使用较长周期过滤噪音，在低波动率环境使用较短周期提高灵敏度。

### 系统结构
- **基础均线**：20日均线
- **波动率指标**：ATR(14)
- **自适应参数**：根据ATR动态调整均线周期

### 自适应公式
```python
# 高波动率时使用较长周期（过滤噪音）
# 低波动率时使用较短周期（提高灵敏度）
adjusted_period = base_period * (1 + ATR / ATR_20day_avg)
```

### 开仓条件
价格上穿自适应均线

### 平仓条件
价格跌破自适应均线
