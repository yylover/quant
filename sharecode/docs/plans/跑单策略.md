## 批量多策略
python sharecode/scripts/scan_trend_params_from_config.py


### 你现在可以直接跑参数扫描

```bash
cd /Users/quickyang/Documents/workspace/develop/mycode/quant
python sharecode/scripts/scan_trend_params_from_config.py
```

它会：

- 从 `sharecode/config.yaml` 读取数据库配置和扫描网格（我已帮你加了 `scan:` 配置段）
- 从 MySQL 读取 **512800 的 OHLC**
- 批量回测这些策略网格：
  - `supertrend`：`atr_window × multiplier`
  - `donchian`：`entry_n × exit_n`
  - `ma_slope`：`window × slope_window × enter/exit_slope`
  - `timing_ma`：`fast × slow`
- 输出终端 Top 20，并保存到：
  - `sharecode/out/scan_512800_trend_params.csv`

### 配置位置（可改网格大小）

你要改扫描范围，直接改 `sharecode/config.yaml` 里的：

- `scan.symbol / scan.exchange / scan.interval`
- `scan.grids.supertrend / donchian / ma_slope / timing_ma`

另外我也顺手修了 SuperTrend 的 `FutureWarning`（boolean fillna downcast）问题。
