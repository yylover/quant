#### 5）10 个 ETF 的动量组合趋势（vectorbt 组合权重）

假设这 10 个 ETF 的 CSV 都在 `sharecode/data` 下：

```bash

python sharecode/scripts/vectorbt_multi_momentum_from_dir.py \
  --csv-dir sharecode/data \
  --glob "*_daily_qfq.csv" \
  --lookback 60 \
  --top-n 5 \
  --rebalance-days 20 \
  --weight-scheme inv_vol \
  --max-weight 0.25 \
  --init-cash 100000 \
  --fees 0.001 \
  --slippage 0.0005

```

这会严格用 vectorbt 的组合权重做回测，输出整个多 ETF 组合的收益、回撤、年化收益、波动和 Sharpe。
