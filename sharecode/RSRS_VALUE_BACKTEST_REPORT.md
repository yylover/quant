**回测区间**: 2018-01-01 ~ 2023-12-31

---

## 一、策略概述

### 1.1 策略逻辑

**RSRS择时（Reference State Regression on Slope）**
- 对最高价和最低价进行滚动回归，计算斜率（beta）代表支撑阻力强度
- 使用标准化方法计算择时信号：`zscore * beta * r2`
- 当RSRS值 > 买入阈值时，市场强势，执行选股
- 当RSRS值 < 卖出阈值时，市场弱势，清仓或保持空仓

**价值选股**
- PB因子：市净率越低越好
- ROE因子：净资产收益率越高越好
- 排名合成：将PB排名和ROE排名按权重合成综合得分

**低质量因子**
- 使用低波动率因子筛选稳定性较高的标的

### 3.1 买入阈值影响

|---------|--------|---------|

**结论**: 买入阈值设置为0.1~0.2之间效果最佳，过低或过高都会减少交易机会。

| 调仓周期 | 收益率 | 调仓次数 |

**结论**: 30日调仓频率效果最佳。

## 四、策略特点

### 4.1 优势

1. **择时保护**: RSRS因子有效识别市场强弱，避免在弱势市场持仓
2. **价值选股**: 结合PB和ROE双重因子，选择低估高质标的
3. **低波动**: 加入低质量因子，提高组合稳定性
4. **风险可控**: 最大回撤仅0.17%

### 4.2 劣势

1. **收益较低**: 在2018-2023年期间，策略收益有限（+0.18%）
2. **信号稀少**: RSRS择时较为严格，交易机会较少
3. **参数敏感**: 买卖阈值对策略效果影响较大
4. **数据依赖**: 价值因子需要真实的PB、ROE数据（当前为模拟）

## 五、对比原始策略

原始聚宽策略（参考）：
- 使用沪深300指数作为基准
- 选股范围：全A股
- 调仓频率：每日判断
- 买入阈值：0.7
- 卖出阈值：-0.7

当前实现与原始策略的差异：
1. **标的池**: 原始策略使用全A股，当前使用32个ETF
2. **价值因子**: 原始策略有真实财务数据，当前为模拟数据
3. **择时逻辑**: 基本一致，使用RSRS标准化信号
4. **调仓频率**: 当前简化为定期调仓

## 六、改进建议

### 6.1 短期改进

1. **接入真实财务数据**: 从Tushare或其他数据源获取PB、ROE数据
2. **优化RSRS参数**: 使用网格搜索或贝叶斯优化寻找最优参数
3. **扩展标的池**: 增加更多ETF或直接使用A股数据
4. **改进择时逻辑**: 考虑结合其他择时指标（如市场情绪、资金流向）

### 6.2 长期改进

1. **机器学习**: 使用ML方法合成因子权重
2. **行业中性**: 引入行业分类，避免行业集中风险
3. **动态调仓**: 根据市场波动率动态调整调仓频率
4. **组合优化**: 使用均值-方差模型优化权重分配
