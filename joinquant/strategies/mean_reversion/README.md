## 策略概述

### 1. 标准均值回归策略 (mean_reversion.py)

**核心原理：**
- 计算N日移动平均线（MA）作为均值
- 计算N日标准差（Std）
- 使用Z-score衡量价格偏离程度：`(当前价 - MA) / Std`

**交易规则：**
- 买入条件：Z-score ≤ -2（价格低于均值2倍标准差）
- 卖出条件：Z-score ≥ 2（价格高于均值2倍标准差）
- 平仓条件：偏离缩小到0.5倍标准差内

**参数配置：**
```python
g.window = 20      # 均线周期
g.num_std = 2.0    # 触发阈值（倍数）
g.revert_ratio = 0.5  # 回归系数
```

**特点：**
- 理论基础扎实，统计学方法严谨
- 适合捕捉明显的均值回归机会
- 需要较大的偏离空间才触发

---

### 2. 布林带策略 (bollinger_bands.py)

**核心原理：**
- 布林带由三条线组成：中轨（MA）、上轨（MA + k×Std）、下轨（MA - k×Std）
- 价格在通道内波动，触及边界时反转

**交易规则：**
- 买入条件：价格触及或跌破下轨
- 卖出条件：价格触及或突破上轨

**参数配置：**
```python
g.window = 20   # 布林带周期
g.num_std = 2.0  # 标准差倍数
```

**特点：**
- 直观易懂，通道清晰
- 适合区间震荡行情
- 与趋势策略形成互补

---

### 3. RSI策略 (rsi_strategy.py)

**核心原理：**
- RSI衡量价格变动的相对强弱
- 取值范围0-100，70以上超买，30以下超卖

**交易规则：**
- 买入条件：RSI < 30（超卖）
- 卖出条件：RSI > 70（超买）

**参数配置：**
```python
g.period = 14       # RSI计算周期
g.oversold = 30     # 超卖阈值
g.overbought = 70   # 超买阈值
```

**特点：**
- 反映短期超买超卖状态
- 对快速反转敏感
- 适合短期波段操作

---

### 4. Z-Score均值回归策略 (zscore_reversion.py)

**核心原理：**
- 使用60日计算周期，比标准版更长
- Z-Score = (当前价 - 均值) / 标准差
- 更适合捕捉中长期的均值回归机会

**交易规则：**
- 买入条件：Z-Score ≤ -2（价格低于均值2倍标准差）
- 卖出条件：Z-Score ≥ 2（价格高于均值2倍标准差）
- 平仓条件：回到 ±1 倍标准差内

**参数配置：**
```python
g.window = 60          # 计算周期
g.k = 2.0              # Z-Score阈值
g.revert_threshold = 1.0  # 回归阈值
```

**特点：**
- 适合中长期震荡行情
- 更稳定的统计特征
- 需要较长的暖机期

---

### 5. 偏离度均值回归策略 (deviation_reversion.py)

**核心原理：**
- 计算价格相对于移动平均线的偏离百分比
- 偏离度 = (当前价 - 均值) / 均值
- 当偏离超过阈值时进行反向交易

**交易规则：**
- 买入条件：偏离度 ≤ -5%（价格低于均线5%）
- 卖出条件：偏离度 ≥ 5%（价格高于均线5%）
- 平仓条件：回到 ±1% 范围内

**参数配置：**
```python
g.ma_window = 20       # 移动平均线周期
g.entry_dev = 0.05     # 入场偏离阈值（5%）
g.exit_dev = 0.01      # 出场偏离阈值（1%）
```

**特点：**
- 参数直观，易于理解
- 响应速度快
- 适合短线操作

---

### 6. KDJ均值回归策略 (kdj_reversion.py)

**核心原理：**
- KDJ指标由K、D、J三条线组成
- K值和D值在0-100之间波动
- 超买超卖状态明确

**交易规则：**
- 买入条件：K ≤ 20 且 D ≤ 20（超卖）
- 卖出条件：K ≥ 80 且 D ≥ 80（超买）

**参数配置：**
```python
g.window = 9          # RSV计算周期
g.m1 = 3              # K值平滑系数
g.m2 = 3              # D值平滑系数
g.low = 20            # 超卖阈值
g.high = 80           # 超买阈值
```

**特点：**
- 反应灵敏
- 适合短线
- 三条线提供确认信号

---

### 7. 威廉指标均值回归策略 (williams_r_reversion.py)

**核心原理：**
- 威廉指标衡量价格在区间中的位置
- 取值范围：0 到 -100
- 接近边界表示极端状态

**交易规则：**
- 买入条件：%R ≤ -80（超卖）
- 卖出条件：%R ≥ -20（超买）

**参数配置：**
```python
g.window = 14       # 计算周期
g.low = -80         # 超卖阈值
g.high = -20        # 超买阈值
```

**特点：**
- 反应极快
- 适合短线操作
- 在震荡市表现良好

---

### 8. CCI均值回归策略 (cci_reversion.py)

**核心原理：**
- CCI衡量价格偏离统计平均水平的程度
- 考虑了典型价格（高+低+收）/3
- 适合识别超买超卖极端情况

**交易规则：**
- 买入条件：CCI ≤ -100（超卖）
- 卖出条件：CCI ≥ 100（超买）

**参数配置：**
```python
g.window = 20       # 计算周期
g.low = -100        # 超卖阈值
g.high = 100        # 超买阈值
```

**特点：**
- 识别极端状态
- 适用于各种时间周期
- 捕捉较大反转机会

---

### 9. MACD均值回归策略 (macd_reversion.py)

**核心原理：**
- MACD柱状图反映动能强弱
- 柱状图正向过大表示超买
- 柱状图负向过大表示超卖

**交易规则：**
- 买入条件：MACD柱状图 ≤ -threshold（负向过大）
- 卖出条件：MACD柱状图 ≥ threshold（正向过大）

**参数配置：**
```python
g.fast = 12            # 快线周期
g.slow = 26            # 慢线周期
g.signal = 9           # 信号线周期
g.threshold = 0.5      # 触发阈值
```

**特点：**
- 结合趋势和动量
- 识别过度拉升或砸盘
- 捕捉动量回归

---

### 10. 乖离率均值回归策略 (bias_reversion.py)

**核心原理：**
- 乖离率衡量价格偏离均线的程度
- BIAS = (当前价 - 均线) / 均线
- 基于价格围绕均线波动的假设

**交易规则：**
- 买入条件：BIAS ≤ -5%（负乖离过大）
- 卖出条件：BIAS ≥ 5%（正乖离过大）
- 平仓条件：回到 ±1% 范围内

**参数配置：**
```python
g.window = 20          # 移动平均线周期
g.low = -0.05          # 超卖阈值（-5%）
g.high = 0.05          # 超买阈值（5%）
g.exit_dev = 0.01      # 离场阈值（±1%）
```

**特点：**
- 计算简单直观
- 参数易于优化
- 适合短线波段

---

### 11. 布林带宽度回归策略 (bb_width_reversion.py)

**核心原理：**
- 带宽 = (上轨 - 下轨) / 中轨
- 衡量市场波动率
- 低波动率后往往伴随高波动

**交易规则：**
- 买入条件：带宽 ≤ 历史分位数10%（波动率压缩）
- 卖出条件：带宽 ≥ 历史分位数50%（波动率恢复）

**参数配置：**
```python
g.window = 20             # 布林带周期
g.n_std = 2.0             # 标准差倍数
g.width_percentile = 0.1  # 带宽分位数
g.width_lookback = 60     # 带宽历史回看期
```

**特点：**
- 捕捉波动率扩张
- 适合横盘突破
- 需要方向判断

---

### 12. ATR均值回归策略 (atr_reversion.py)

**核心原理：**
- ATR衡量真实波幅
- ATR通道：均线 ± k × ATR
- 突破通道表示过度延伸

**交易规则：**
- 买入条件：价格 ≤ 均线 - 2 × ATR（跌破下轨）
- 卖出条件：价格 ≥ 均线 + 2 × ATR（突破上轨）

**参数配置：**
```python
g.window = 14             # ATR计算周期
g.ma_window = 20          # 均线周期
g.atr_multiplier = 2.0    # 入场ATR倍数
g.exit_multiplier = 0.5   # 离场ATR倍数
```

**特点：**
- 考虑跳空情况
- 基于真实波动
- 适应波动率变化

---

### 13. ROC均值回归策略 (roc_reversion.py)

**核心原理：**
- ROC衡量价格在N日内的变化幅度
- ROC = (当前价 - N日前价) / N日前价
- 反映涨跌速度和方向

**交易规则：**
- 买入条件：ROC ≤ -8%（跌幅过大）
- 卖出条件：ROC ≥ 8%（涨幅过大）
- 平仓条件：回到 ±2% 范围内

**参数配置：**
```python
g.window = 12          # ROC计算周期
g.low = -0.08          # 超卖阈值（-8%）
g.high = 0.08          # 超买阈值（8%）
g.exit_dev = 0.02      # 离场阈值（±2%）
```

**特点：**
- 反应速度极快
- 适合短线
- 需要止损保护

---

### 14. PSAR均值回归策略 (psar_reversion.py)

**核心原理：**
- PSAR是趋势跟踪指标
- 计算价格相对于PSAR线的偏离
- 使用标准差量化偏离程度

**交易规则：**
- 买入条件：价格 < PSAR - 1.5 × std（过度偏离下方）
- 卖出条件：价格 > PSAR + 1.5 × std（过度偏离上方）

**参数配置：**
```python
g.af = 0.02            # 加速因子初始值
g.max_af = 0.2         # 最大加速因子
g.lookback = 3         # 计算偏离度的回看期
g.k = 1.5              # 偏离阈值倍数
```

**特点：**
- 结合趋势信息
- 识别过度延伸
- 计算相对复杂

## 风险提示

### 共同风险
1. **趋势市亏损**：在单边牛市或熊市中，价格持续偏离，策略会不断亏损
2. **假信号**：超买超卖可能持续存在，导致过早入场或离场
3. **滑点成本**：频繁交易会增加手续费和滑点成本

### 各策略特有风险
- **标准均值回归**：需要较大偏离才触发，可能错过小波段机会
- **布林带**：通道收窄时信号减少，通道扩大时假信号增多
- **RSI**：在强趋势行情中容易产生持续的超买超卖假信号

## 改进建议

### 1. 趋势过滤
```python
# 只在震荡行情中启用均值回归
ma_short = close.rolling(20).mean()
ma_long = close.rolling(60).mean()
is_range_bound = abs(ma_short - ma_long) / ma_long < 0.05  # 5%以内的波动
```

### 2. 动态参数
```python
# 根据ATR（平均真实波幅）动态调整阈值
atr = calculate_atr(close, 14)
dynamic_threshold = base_threshold * (atr / atr_ma)
```

### 3. 多周期确认
```python
# 同时满足日线和周线信号
daily_signal = check_daily_rsi()
weekly_signal = check_weekly_rsi()
if daily_signal and weekly_signal:
    execute_trade()
```

### 4. 仓位管理
```python
# 根据偏离程度动态调整仓位
if abs(z) > 3:
    position_ratio = 0.95  # 极端偏离，大仓位
elif abs(z) > 2:
    position_ratio = 0.65  # 中等偏离，中等仓位
else:
    position_ratio = 0.30  # 轻度偏离，小仓位
```

### 5. 止损保护
```python
# 设置固定止损比例
stop_loss = entry_price * 0.95  # 5%止损
if current_price <= stop_loss:
    close_position()
```

## 回测建议

### 参数优化
- 在历史数据上优化参数时，注意避免过拟合
- 建议使用滚动窗口验证和样本外测试
- 参数应具有经济学意义，而非纯粹数据挖掘

### 评估指标
- 年化收益率
- 最大回撤
- 夏普比率
- 胜率与盈亏比
- 交易频率与成本

### 对比基准
- 沪深300指数
- 持有标的ETF
- 其他同类策略
