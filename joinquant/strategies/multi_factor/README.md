# 多因子选股策略包

本目录包含基于多因子选股原理实现的聚宽交易策略。

## 策略列表

| 策略文件 | 策略名称 | 核心因子 | 持股数 | 特点 |
|---------|---------|---------|--------|------|
| multi.py | PB+ROE双因子 | PB、ROE | 10只 | 经典价值投资+动量择时 |
| three_factor.py | 三因子扩展 | 市值、价值、动量 | 15只 | 基于Fama-French模型 |
| growth_value.py | 成长价值复合 | 成长、价值、质量 | 12只 | 成长性+估值+盈利质量 |
| low_vol_quality.py | 低波质量 | 低波、质量、防御 | 15只 | 防御性强，回撤小 |
| sector_rotation.py | 行业轮动 | 行业动量、估值、景气度 | 12只 | 捕捉行业轮动机会 |

## 详细说明

### 1. PB+ROE双因子策略 (multi.py)

**策略思路**：
- 价值因子：PB（市净率），低估值
- 质量因子：ROE（净资产收益率），高盈利
- 择时因子：RSRS（相对强弱回归斜率）

**适用场景**：
- 震荡偏牛市
- 价值投资风格
- 能承受一定回撤

**历史回测表现**：
- 总收益：426.40%
- 年化收益：15.04%
- 最大回撤：34.62%
- 夏普比率：0.610

---

### 2. 三因子扩展策略 (three_factor.py)

**策略思路**：
- 市值因子（SMB）：小盘股往往有超额收益
- 价值因子（HML）：低估值股票收益更好
- 动量因子（MOM）：过去强势股票延续强势

**因子权重**：
- 市值：30%
- 价值：30%
- 动量：40%

**适用场景**：
- 小盘股机会较多
- 市场动量明显
- 风格轮动频繁

---

### 3. 成长价值复合策略 (growth_value.py)

**策略思路**：
- 成长因子：营收增长、利润增长、ROE增长
- 价值因子：PE、PB、PEG（动态估值）
- 质量因子：ROE、ROIC、毛利率

**因子权重**：
- 成长：40%
- 价值：30%
- 质量：30%

**适用场景**：
- 成长风格明显的市场
- 选成长性好估值合理的公司
- 中长期持有

---

### 4. 低波质量策略 (low_vol_quality.py)

**策略思路**：
- 低波因子：历史波动率、Beta、最大回撤
- 质量因子：ROE、ROA、毛利率、债务/权益比
- 防御因子：股息率、现金流/净利润

**因子权重**：
- 低波：40%
- 质量：30%
- 防御：30%

**适用场景**：
- 熊市或高波动市场
- 追求稳健收益
- 适合保守投资者

**特点**：
- 防御性强
- 回撤小
- 现金流好

---

### 5. 行业轮动策略 (sector_rotation.py)

**策略思路**：
- 行业动量因子：行业指数收益率、RSI突破
- 行业估值因子：行业PE分位数、PB分位数
- 行业景气度因子：行业营收增长、订单增速

**持仓结构**：
- 配置3个优势行业
- 每个行业选4只股票
- 总持仓12只股票

**适用场景**：
- 行业轮动明显的市场
- 风格切换频繁
- 中短线操作

**特点**：
- 捕捉行业轮动
- 动量驱动
- 估值保护

## 共同特点

### 择时机制

所有策略都使用 RSRS（相对强弱回归斜率）择时：

```python
RSRS计算流程：
1. 滚动回归：High = a + b × Low（N日窗口）
2. 标准化：Z-score = (β当前 - μ历史) / σ历史
3. 右偏修正：修正后RSRS = Z-score × β × R²
```

**交易规则**：
- 买入：修正后RSRS > 0.7 → 选股买入
- 卖出：修正后RSRS < -0.7 → 清仓离场
- 区间内：保持原有持仓

### 风险控制

- 分散持仓：10-15只股票等权重
- 择时保护：RSRS卖出信号时清仓
- 基本面过滤：剔除亏损股、ST股、数据异常股

## 风险提示

### 共同风险

1. **熊市风险**：大部分多因子策略在熊市可能失效
2. **择时滞后**：RSRS在急跌时可能滞后
3. **因子拥挤**：多因子策略使用增多导致效果下降

### 特定风险

- **PB+ROE**：价值陷阱、行业集中
- **三因子**：小盘股流动性差、动量失效
- **成长价值**：成长陷阱、估值风险
- **低波质量**：收益有限、牛市跑输
- **行业轮动**：轮动失败、滞后性

## 改进方向

### 选股层面

1. **增加因子**：PE、市值、动量等20+因子
2. **因子权重优化**：基于IC、ICIR
3. **因子正交化**：行业、市值中性化
4. **机器学习**：XGBoost、LightGBM

### 择时层面

1. **多信号融合**：MA60、ATR、VIX等
2. **动态阈值**：根据市场波动率调整
3. **止损机制**：单票止损、组合止损
4. **仓位管理**：根据RSRS值动态调整

### 交易层面

1. **降低换手**：设置最小持仓天数
2. **分批建仓**：分3天买入
3. **行业分散**：限制单行业持仓比例
4. **再平衡**：定期再平衡
