
# 基础

## 市场

### T+1（交割与卖出）
含义：当天买入的证券，当天不能卖出；要 下一个交易日 才能卖。

### 涨跌停（价格限制）
含义：在一个交易日内，成交价相对前一交易日收盘价，只能在规定幅度内波动，触及上限叫「涨停」，下限叫「跌停」。

### 3. 费用（佣金、印花税、过户费等）
股票（A 股场内）
* 佣金：买卖双向，费率由券商约定（常见万 2～万 3 量级），有最低 5 元 等规定很常见。
* 印花税：目前 卖出 时征收（买入不收），税率以国家现行政策为准（常见表述为成交金额的 千分之 0.5 等，以最新财税规则为准）。
* 过户费：沪市股票有（费率很低）；深市规则历史上略有不同，以交易所/券商公示为准。

### 4. 复权（前复权、后复权、不复权）
为什么要复权：分红、送股、配股后，股价会 跳空，K 线不连续。复权是把历史价格 调整，使曲线反映「长期涨跌」而不是被除权缺口打断。

前复权： 以最新价为基准，向前调整历史价

## 风险
### 1. 回撤（Drawdown）
含义：净值从阶段高点往下掉多少，通常用百分比表示。

### 2. 仓位（Position / 仓位管理）
含义：你有多少钱暴露在风险里——现金 vs 股票/ETF 市值的比例，或多标的时每只占多少。
 「以损定量」：先定「这笔最多亏本金的百分之几」，再反推该买多少股——本质是把仓位和风险绑在一起。

### 3. 止损（Stop Loss）
含义：事先约定：价格（或亏损）走到某一步就认错离场，避免小亏拖成大亏。
    * 固定比例止损：例如跌破买入价 5% 就卖。
    * 技术止损：跌破某根均线、前低、ATR 倍数等。
    * 时间止损：持有 N 天仍不达预期就平掉。

### 4.  回撤、仓位、止损三者的关系（一句话）
仓位决定你参与多少风险；
止损决定你单笔/单段错的时候最多付出多少；

### 5. 为什么「回测赚钱 ≠ 未来赚钱」
回测只是在过去数据上验证规则，未来市场会变，常见原因包括：

| 原因 | 说明 |
|------|------|
| **过拟合** | 参数、规则是为某一段行情「调出来的」，换一段就失效 |
| **市场机制变化** | 政策、结构、波动率、参与者行为变了 |
| **成本与成交** | 回测里手续费、滑点、涨停买不进等往往比实盘乐观 |
| **幸存者偏差** | 只看见「活下来」的标的或策略 |
| **未来函数** | 回测里不小心用了「当时还不知道」的数据（实盘做不到） |

## 统计

### 1. 均值
移动均值怎么计算的?
**含义**：一组数的平均水平。常用 **算术平均**：所有值相加 ÷ 个数。

- **移动平均线（MA）**：收盘价的滚动均值，用来表示「近期平均成本/中枢」。
- **策略评估**：年化收益、日均收益等，本质也是对收益序列做平均（或年化处理）。

**注意**：均值**容易被极端值拉偏**；收益分布若「偶尔大赚、经常小亏」，只看平均收益会误判稳定性。

### 2. 标准差（Standard Deviation）

**含义**：数据相对均值**分散程度**的度量。标准差大 → 波动大；小 → 更集中。

**在量化里**：

- **波动率**：常对**收益率**算标准差（再年化），衡量策略或标的有多「抖」。
- **布林带**：中轨 ± k×标准差 形成上下轨。
- **Z-score / 均值回归**：\((价格 - 均值) / 标准差\)，看偏离「典型波动」有多少倍。


### 2. 标准差（Standard Deviation）

**含义**：数据相对均值**分散程度**的度量。标准差大 → 波动大；小 → 更集中。

**在量化里**：

- **波动率**：常对**收益率**算标准差（再年化），衡量策略或标的有多「抖」。
- **布林带**：中轨 ± k×标准差 形成上下轨。
- **Z-score / 均值回归**：\((价格 - 均值) / 标准差\)，看偏离「典型波动」有多少倍。

**注意**：标准差假设波动结构相对稳定；危机期、结构突变时，**历史标准差会低估未来风险**。


### 相关性（Correlation）

**含义**：两个变量**一起涨落**的线性关联程度，常用 **皮尔逊相关系数**，介于 **-1 到 1**。

- 接近 **+1**：同向变动强  
- 接近 **-1**：反向变动强  
- 接近 **0**：线性关系弱（不代表没有关系，可能有非线性关系）

**在量化里**：

- **分散化**：选相关性低的标的组合，降低组合波动。
- **因子/策略**：看因子与收益、策略与基准的相关性。
- **多重共线性**：多个因子彼此高度相关时，回归/组合权重会不稳定，要谨慎。

**注意**：相关性**随时间会变**；过去低相关，未来可能一起跌（尾部相关性）。


### 4. 过拟合（Overfitting）

**含义**：模型或规则**太贴合历史数据里的噪声**，把偶然当成规律，**换一段数据或实盘就变差**。

5. 样本外检验（Out-of-Sample Testing）

**含义**：在**模型/参数没有「见过」的数据**上检验表现，看是否只是记住了历史。

**常见做法**：

| 方式 | 做法 |
|------|------|
| **时间切分** | 用前 70% 时间调参或训练，后 30% 只做一次检验，不再改参 |
| **训练 / 验证 / 测试** | 训练定模型，验证选超参，**测试集只最后用一次** |
| **滚动 / 走样本外（Walk-forward）** | 不断用过去窗口拟合，在紧接着的一小段上检验，多段拼接看稳定性 |

# 常见策略的原理


## 均线
- **简单移动平均 SMA**：  
  \[
  MA_t = \frac{1}{N}\sum_{i=0}^{N-1} P_{t-i}
  \]
  最常用，聚宽里 `rolling().mean()` 一类就是这类思路。

- **指数移动平均 EMA**：近期权重更大，对价格变化更敏感、更「跟得紧」，MACD 里常用 EMA。

2. 均线策略的核心思想（原理）

1. **趋势**：价格在均线一侧持续运行，常被视为「短期平均成本」相对「长期平均成本」占优，即趋势方向。  
2. **支撑 / 压力（经验规则）**：上涨中，均线有时被当作回调支撑；下跌中，均线有时当作反弹压力（不是定律，只是常见用法）。  
3. **金叉 / 死叉**：  
   - **短均线上穿长均线** → 常称 **金叉**，信号上偏多（认为趋势可能转多或延续）。  
   - **短均线下穿长均线** → **死叉**，信号上偏空或离场。

## **MACD**（Moving Average Convergence Divergence，指数平滑异同移动平均线）不是一条线，而是一套指标，常用来描述：

- **趋势方向**（多空谁占优）  
- **趋势强弱变化**（加速还是减速）  
- **可能的拐点**（金叉/死叉、柱线由正转负等）

它建立在 **指数移动平均（EMA）** 上，所以比简单均线 **更敏感**，但仍带 **滞后**。


## 3. 常见用法（原理层面）


1. **快线 EMA**：周期常用 **12**  
2. **慢线 EMA**：周期常用 **26**  
3. **DIF（差离值）**  
   \[
   DIF = EMA_{12}(收盘) - EMA_{26}(收盘)
   \]  
   可理解成：**短期趋势相对长期趋势偏离多少**。DIF > 0 偏多，DIF < 0 偏空。

4. **DEA（信号线）**  
   \[
   DEA = EMA_9(DIF)
   \]  
   对 DIF 再平滑，用来和 DIF 做 **交叉** 信号。

5. **MACD 柱（柱状图）**  
   常见写法：  
   \[
   \text{柱} = 2 \times (DIF - DEA)
   \]  
   （有的软件不写 2 倍，只是缩放不同，**形状一致**。）  
   - 柱 > 0：DIF 在 DEA 上方，多头动能相对强  
   - 柱 < 0：空头动能相对强  
   - 柱由负转正 / 由正转负：常当作 **动能切换**


| 用法 | 含义 |
|------|------|
| **DIF 上穿 DEA（金叉）** | 短期动能相对长期平滑线走强，**偏多信号** |
| **DIF 下穿 DEA（死叉）** | **偏空或离场信号** |
| **DIF 与 0 轴** | DIF > 0 多在「长期均线之上」的语境里偏多；< 0 偏空（和参数 12/26 有关） |
| **柱线放大/缩小** | 柱变长：趋势加速；柱缩短：趋势减速，可能接近变盘 |
| **背离（进阶）** | 价格新高而 MACD 不新高 → 上涨动能减弱（**仅作预警，不是单独圣杯**） |

本质：**用两条 EMA 的差及其平滑，把「趋势 + 动量」压成几条线和柱**，比单看一根均线信息多一点，但仍是 **滞后指标**。

- 双均线：直接比 **价与均线** 或 **短均线与长均线**。  
- MACD 的 DIF 本质是 **两条 EMA（12 与 26）的差**，再对差做 9 日 EMA 得到 DEA。  
- 所以 **MACD 金叉 ≈ 某种意义上的「快慢趋势差」由弱转强**，和 **均线金叉** 常常 **同向**，但 **触发时间不完全相同**（MACD 多一层平滑，有时更晚，有时过滤掉部分毛刺）。


## 3. 布林

布林带由 **三条线** 组成，画在价格 K 线周围：

- **中轨**：一般是 **N 日收盘价简单移动平均（SMA）**，和「均线」是同一种东西。  
- **上轨**：中轨 + **k × 标准差**（常用 k=2）。  
- **下轨**：中轨 − **k × 标准差**。

含义：用「均线」当中枢，用「波动率（标准差）」当宽度，**通道随波动自动变宽变窄**。

---

## 2. 怎么算（核心公式）

设收盘价为 \(P\)，窗口为 \(N\)：

- 中轨：\(\;MA_t = \text{mean}(P_{t-N+1},\ldots,P_t)\)  
- 标准差：\(\;\sigma_t = \text{std}(P_{t-N+1},\ldots,P_t)\)  
- 上轨：\(\;MA_t + k \cdot \sigma_t\)  
- 下轨：\(\;MA_t - k \cdot \sigma_t\)  

**k** 越大，通道越宽，触及上下轨越少；**N** 越大，中轨越平滑，通道反应越慢。

你仓库里 **`bollinger_bands.py`**、**`mean_reversion/bollinger_bands.py`** 等就是这类逻辑的具体实现。

## 3. 常见用法（原理）

| 思路 | 说明 |
|------|------|
| **均值回归** | 价格跌到 **下轨附近** 认为短期偏「超卖」，买入；涨到 **上轨附近** 卖出。假设：多数时候价格在通道内波动，极端触及边界会往回摆。 |
| **突破** | 价格 **有效突破上轨** 并放量，当作趋势加强（与均值回归相反）；跌破下轨当作弱势延续。同一指标，两种哲学，**不能混用同一套规则不说明白**。 |
| **通道收窄** | \(\sigma\) 很小 → 波动率低，常有人说「酝酿变盘」；需配合方向确认，单独用容易误判。 |
| **带宽** | \((上轨−下轨)/中轨\) 有时单独当「波动率指标」用（你仓库里有 `bb_width_reversion` 一类变体）。 |

## 4. 和均线、MACD 的关系

- **中轨 = 均线**：布林带是 **「均线 + 波动率」** 的可视化。  
- **MACD**：看趋势与动能；布林带看 **相对中枢的偏离程度**。  
- 常组合：**大趋势用均线/MACD，小区间用布林带做加减仓**（逻辑上要自洽）。


## 3. 动量是什么（核心思想）

**动量**：价格（或收益）在一段时间里 **持续上涨或下跌**，短期内往往有 **惯性**——涨过的还可能再涨一段，跌过的还可能再跌一段。

**动量策略**：相信这种惯性，**买「过去一段时间涨得多」的，卖或回避「跌得多」的**；或简化为 **单标的：过去 N 日涨 → 继续持多/买入，转弱则离场**。

这和 **均线趋势** 有关，但更直接看 **收益率**，不一定先看均线交叉。

## 2. 常见怎么算

**时间序列动量（一个标的）**

- \(N\) 日收益率：  
  \[
  R = \frac{P_t}{P_{t-N}} - 1
  \]
- 规则举例：\(R > 0\) 且价格在均线上方 → 做多；\(R \le 0\) 或跌破均线 → 平仓。

**截面动量（多只股票）**

- 对每只股票算过去 \(N\) 日（或 6、12 月）收益，**全市场排序**，买入 **前 10%～20%**，定期调仓；或做空后段（视市场是否允许）。

你仓库里的 **`strategies/trend/momentum.py`** 是 **单标的 + N 日收益 + 均线过滤** 的典型写法。

## 3. 和均线、MACD 的区别（一句话）

| 类型 | 关注点 |
|------|--------|
| **均线** | 价格相对「平均成本」的位置与交叉。 |
| **MACD** | 两条 EMA 的差及其变化（趋势 + 动能）。 |
| **动量** | **赤裸裸看一段区间涨跌幅**，再可加均线等过滤。 |

动量可以 **不用均线**；加上均线是为了 **减少震荡里假信号**。

## 4. RSI 是什么

RSI 把一段时间里 **上涨幅度** 和 **下跌幅度** 做个对比，压到 **0～100** 之间，用来描述：

- 近期是 **涨得多还是跌得多**（多空力量对比）  
- 是否进入常识里的 **超买 / 超卖** 区（注意：超买还可以再涨，超卖还可以再跌）

属于 **振荡类指标**，常和均线、趋势过滤一起用。

---

## 2. 怎么算（经典公式）

对收盘价做逐日涨跌：

- 涨幅：\(\Delta_t = P_t - P_{t-1}\)，只取 **涨的部分**（跌日记为 0）  
- 跌幅：只取 **跌的部分**（涨日记为 0）

在窗口 **N** 日（常用 **14**）内：

- 平均涨幅 \(AU\)：N 日涨幅的平均（常用简单平均或平滑）  
- 平均跌幅 \(AD\)：N 日跌幅的平均（取绝对值）

\[
RS = \frac{AU}{AD} \quad (\text{若 } AD=0 \text{ 需单独处理，避免除零})
\]
\[
RSI = 100 - \frac{100}{1 + RS}
\]

**直觉**：

- 若 N 天里 **几乎都在涨**，\(AD\) 很小 → \(RS\) 很大 → **RSI 接近 100**  
- 若 **跌得多**，\(AU\) 小、\(AD\) 大 → **RSI 接近 0**

---

## 3. 常见怎么用（策略逻辑）

| 用法 | 说明 |
|------|------|
| **超买超卖** | RSI **高于 70～80** 视为超买，偏卖出/减仓；**低于 20～30** 视为超卖，偏买入。阈值可按品种和周期改。 |
| **中线 50** | RSI **上穿 50** 偏多，**下穿 50** 偏空（一种简化趋势过滤）。 |
| **背离（进阶）** | 价格新高而 RSI 不新高 → 上涨动能减弱（仅作预警，需配合其他规则）。 |


## KDJ
 KDJ 是什么

在 **N 日** 区间内，看 **当前收盘价** 在 **最高价～最低价** 之间的相对位置，再平滑成 **K、D** 两条线，并派生出 **J**。  
用来描述：**短期是偏强还是偏弱、是否进入常识上的超买/超卖**。

- **K、D**：在 0～100 附近波动（具体实现略有差异，常见 K、D 在 0～100）。  
- **J**：对 K、D 的放大，波动更大，可超出 0～100。

---

## 2. 怎么算（常见公式思路）

设最近 **N** 根 K 线（常用 **9**）：

- **RSV（未成熟随机值）**  
  \[
  RSV_t = \frac{C_t - L_n}{H_n - L_n} \times 100
  \]  
  \(C_t\)：今日收盘；\(H_n\)、\(L_n\)：N 日内最高、最低。  
  若 \(H_n = L_n\)（横盘），需按软件规则处理（避免除零），常令 RSV 为 50 或沿用前值。

- **K 值**：对 RSV 做平滑（常类似 EMA/SMA，如 **K = 2/3 × 前K + 1/3 × RSV**）。  
- **D 值**：对 K 再平滑（如 **D = 2/3 × 前D + 1/3 × K**）。  
- **J 值**：  
  \[
  J = 3K - 2D
  \]  
  所以 J 波动比 K、D **更剧烈**，先到极端区的情况更多。

不同行情软件 **平滑系数** 可能略有不同，回测时要和 **聚宽/你的数据源** 实现一致。

---

## 3. 常见怎么用（策略逻辑）

| 用法 | 说明 |
|------|------|
| **超买超卖** | K、D **大于 80～90** 偏超买，**小于 10～20** 偏超卖（阈值可调）。 |
| **金叉 / 死叉** | **K 上穿 D** 偏多，**K 下穿 D** 偏空（尤其在超卖区金叉、超买区死叉，被说得较多）。 |
| **J 线** | J **>100 或 <0** 常被视为短期过热或过冷（更敏感、假信号也更多）。 |

和 RSI 一样：**强趋势里 K、D 可以长期在高位或低位**，不能机械认为「超买必跌」。
